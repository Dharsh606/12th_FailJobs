# 12th Fail Jobs 🚀

**Empowering Every Job Seeker - No Qualification Barriers**

## 📋 About

12th Fail Jobs is a comprehensive job portal platform designed to connect job seekers with employment opportunities, regardless of their educational qualifications. Our mission is to bridge the gap between employers and talented individuals who may not have traditional academic credentials but possess valuable skills and dedication.

## 🚀 Getting Started

### Prerequisites
- **Node.js** 14.0 or higher
- **npm** 6.0 or higher
- **Web browser** (Chrome, Firefox, Safari, Edge)

### Quick Start (Recommended)

1. **Download the project**
   ```bash
   # Clone or download the project
   cd 12thfailjob
   ```

2. **Run the quick start script**
   ```bash
   # Windows
   start.bat
   
   # Or manually:
   npm install
   node server/index.js
   ```

3. **Access the application**
   - Open your browser
   - Navigate to: `http://localhost:3001`

### 🚀 Vercel Deployment (Production)

1. **Deploy to Vercel**
   - Visit [vercel.com](https://vercel.com)
   - Import your GitHub repository: `Dharsh606/12th_FailJobs`
   - Click **Deploy** (no configuration needed)
   - Your app will be live at: `https://12th-fail-jobs.vercel.app/`

2. **Automatic Updates**
   - Every push to GitHub triggers automatic deployment
   - Zero downtime updates
   - Global CDN distribution

### Manual Installation

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the server**
   ```bash
   node server/index.js
   ```

3. **Access the application**
   - Open browser to: `http://localhost:3001`
